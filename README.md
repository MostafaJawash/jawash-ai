# Jawash AI - تطبيق سؤال وجواب

تطبيق React بسيط، **Jawash AI**، بيسمحلك تكتب سؤالك وتشوف رده في واجهة شات. (مبني تقنيًا على Gemini API من Google.)

## المتطلبات
- Node.js (نسخة 18 أو أحدث) — تقدر تتأكد بكتابة `node -v` في التيرمينال
- مفتاح Gemini API (اتعمل بالفعل في الخطوات السابقة)

## خطوات التشغيل

### 1. فك ضغط المشروع وادخل المجلد
```bash
cd jawash-ai
```

### 2. ثبّت المكتبات المطلوبة
```bash
npm install
```

### 3. أضف مفتاح الـ API
انسخ ملف `.env.example` وسمّيه `.env`:
```bash
cp .env.example .env
```
افتح ملف `.env` وحط مفتاحك بدل النص:
```
VITE_GEMINI_API_KEY=AIzaSy...........................
```

### 4. شغّل المشروع محليًا
```bash
npm run dev
```
هيديك رابط زي `http://localhost:5173` — افتحه في المتصفح.

## هيكل المشروع
```
jawash-ai/
├── index.html          # نقطة الدخول HTML
├── src/
│   ├── main.jsx        # نقطة دخول React

├── .env.example        # نموذج لملف المتغيرات البيئية

```

## ملاحظات مهمة
- **الأمان**: هذا المشروع يستدعي API مباشرة من المتصفح، وده مناسب للتجربة والتعلم فقط. أي حد يفتح أدوات المطور (DevTools) في المتصفح هيقدر يشوف مفتاحك. لمشروع حقيقي/إنتاجي، لازم تعمل خادم خلفي (backend) بسيط يخبي المفتاح ويكون هو اللي يتواصل مع Gemini.
- **الموديل المستخدم**: `gemini-flash-latest` وهو سريع ومجاني ضمن حدود المستوى المجاني.
- ملف `.env` متضاف في `.gitignore` عشان ميترفعش بالغلط لو استخدمت Git/GitHub.
